# Free Games & More Here

A fun arcade-style gaming website with 20+ mini games, player signups, and an admin dashboard.

## Features

- 🎮 20 playable arcade mini-games
- 📝 Player signup form with email collection
- 📊 Owner admin dashboard with player tracking
- 💾 Local browser storage for player data
- 🎨 Modern, responsive design
- 🚀 Ready for Vercel deployment

## Getting Started

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Play games and test the signup form

### Admin Access
- Username: Owner area at `/index.html#owner`
- Password: `HannesTolmay2026` (demo)

## Deployment to Vercel

1. Push to GitHub (already done!)
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Select your GitHub repo `free-games-arcade`
5. Deploy!

Your site will be live at: `your-project.vercel.app`

## Asset Files Needed

- `assets/hero-games.png` - Hero section background image
- `assets/hannes-arcade-logo.svg` - Logo (included)

## Notes

- Player signups are stored in browser localStorage (demo)
- For production, connect to a database or email service
- Update the share link in the "Momentum" section to point to your domain
- Replace demo passcode with secure authentication
